The TI-99/4A

The TI-99/4A is a home computer from 1981. It features a TMS9900 CPU running at 3 MHz, a TMS9918A video display processor with 16K RAM, a real keyboard, a cartridge slot, and a cassette tape interface.

The firmware and BASIC interpreter are contained in 8K ROM and 18K GROM, but only 256 bytes of RAM are available to the CPU, so the vacant video RAM is used for storing BASIC programs. Most commercial software was provided on ROM/GROM cartridges.

The TI-99/4A can be expanded in numerous ways, and in this emulator a 32K expansion RAM and 3 floppy drives are available by default.
